### ทำไมถึงเกิด Project นี้?

ซึ่งสามารถใช้ได้สูงสุด 6 เทอม(ณ เวอร์ชั่น 7) {ดูได้ที่แถบด้านบน} และเป็นภาษาอังกฤษอีกด้วย<br>
